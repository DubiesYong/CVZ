# 🚀 Cat vs Zombie (CVZ)

A space-themed infinite survival shooter built with **Python (Pygame)** and powered by **Firebase**.

> 🎮 Log in from the website portal to download the game and customize your cat skin.

---

## 🌌 Features

- 🐱 Customizable Cat (color sync from website)
- 👂 Cat ears & tail design
- 💥 Shooting system with cooldown
- 🛡 Shield after taking damage
- ❤️ 3 Lives system
- 📈 Infinite survival mode
- ⚡ Difficulty scales over time
- ☁ Online score saving (Firestore)
- 🔐 Firebase Authentication login system

---

## 🌐 Web Portal

The website allows:

- 🔐 Sign Up / Log In
- 🎨 Change Cat Skin Color
- ⬇ Download CVZ Game
- 🔒 Secure Logout

All player data is stored securely in Firebase Firestore.
